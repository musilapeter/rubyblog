## Overview
A premium, free, open-source developer education platform designed to take learners from their first line of HTML to architecting intelligent AI agents. Built around a unique concept—**The Eras of Technology**—SaneGenius guides learners through the physical and logical evolution of software engineering across 6 progressive learning paths (120+ lessons, 36 projects).

## Key Features & Stack
- **Next.js 16 (App Router)**: Server-side rendering with the latest App Router for instant page loads and deep SEO optimization.
- **TypeScript**: Full type safety across the entire codebase for robust, maintainable code.
- **Tailwind CSS v4**: Glassmorphism aesthetics and premium UX with the latest utility-first CSS framework.
- **Supabase**: Auth and real-time data sync for seamless user progress tracking across devices.
- **Lucide React**: Clean, consistent iconography throughout the platform.

## The Eras of Technology
The platform covers 6 progressive learning paths:
1. **Foundations** — HTML, CSS, and core web fundamentals
2. **Python/Django** — Backend development and server-side logic
3. **JavaScript/React** — Modern frontend frameworks and state management
4. **Backend Systems** — Advanced architecture, APIs, and databases
5. **AI/ML** — Machine learning fundamentals and model deployment
6. **Agentic AI** — Building intelligent, autonomous AI agents

## The Technical Challenge
Engineering a platform that balances deep educational content with premium user experience. The architecture needed to support 120+ lessons with real-time progress tracking, interactive code examples, and a cohesive learning journey that adapts to each learner's pace—all while maintaining instant page loads and glassmorphism aesthetics.
